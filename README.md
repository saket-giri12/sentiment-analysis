 Sentiment Analysis Using Python

This project performs sentiment analysis on customer reviews using Python. It analyzes the text and classifies each review as Positive, Negative, or Neutral. The project also includes data cleaning, exploratory data analysis (EDA), visualizations, and an analytical report to understand customer opinions.



- Load customer review dataset
- Clean and preprocess data
- Perform sentiment analysis using TextBlob
- Classify reviews into Positive, Negative, and Neutral
- Visualize results using charts
- Generate useful insights

## Technologies Used

- Python
- Pandas
- NumPy
- TextBlob
- Matplotlib
- Seaborn
